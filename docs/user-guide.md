# User Guide: Noodles Crypto Dashboards

## Overview

This guide explains how to access and use the **Noodles Crypto Analytics dashboards** for market insights, performance tracking, and data-driven decision-making.

The dashboards are powered by a centralized data pipeline and provide near real-time visibility into cryptocurrency trends, social engagement, and market activity.

---

## Accessing Dashboards

### Option 1: Power BI Service (Recommended)

1. Visit: https://app.powerbi.com
2. Sign in with your organizational account
3. Navigate to **"Noodles Analytics"** workspace
4. Open the **"Executive Dashboard"** report

---

### Option 2: Power BI Desktop

1. Download the `.pbix` file from the shared repository
2. Open using Power BI Desktop
3. Click **"Refresh"** to load the latest data

---

## Available Dashboards

### 1. Executive Dashboard

**Purpose**:
Provides a high-level overview of the cryptocurrency market for executives and decision-makers.

**Key Metrics**:

* Total Market Capitalization
* Average Price Across Currencies
* 24h Trading Volume
* Number of Active Currencies

**How to Use**:

* Use the **date slicer** (top-right) to adjust the time range
* Click on any visual to apply **cross-filtering**
* Hover over charts for detailed tooltips

---

### 2. Top Performers Report

**Purpose**:
Identify top-performing and underperforming cryptocurrencies.

**Features**:

* Top Gainers (sorted by % price change)
* Top Losers
* Volume leaders
* Trend indicators

**How to Use**:

* Adjust the **Top N filter** (default: 10)
* Right-click a currency → **Drill-through** for deeper insights
* Export data via the **"..." menu**

---

### 3. Currency Deep Dive

**Purpose**:
Analyze individual cryptocurrencies in detail.

**Features**:

* Price trends with moving averages
* Volume analysis
* Social engagement metrics (Twitter, Reddit)
* Historical comparisons

**How to Use**:

* Access via **drill-through** from other dashboards
* Use the **currency selector dropdown**
* Compare multiple currencies using **Ctrl + Click**

---

## Common Tasks

### Filter by Date Range

1. Locate the date slicer (top-right)
2. Click the calendar icon
3. Select start and end dates
4. Or use presets like **Last 30 Days** or **Year-to-Date (YTD)**

---

### Export Data

1. Click the **"..." (More options)** menu on any visual
2. Select **"Export data"**
3. Choose format: `.xlsx` or `.csv`
4. Open in Excel for further analysis

---

### Print / Export Report

1. Go to **File → Export → PDF**
2. Select required pages
3. Adjust layout if needed
4. Save the file

---

### Share with Colleagues

1. Click **"Share"** (top-right)
2. Enter recipient email addresses
3. Set permissions (View / Edit)
4. Add message and send

---

## Understanding Key Metrics

| Metric         | Description                         | Business Use                |
| -------------- | ----------------------------------- | --------------------------- |
| Market Cap     | Total value of all cryptocurrencies | Measure overall market size |
| 24h Volume     | Trading activity in last 24 hours   | Identify liquidity          |
| Price Change % | Percentage price movement           | Spot trends and momentum    |
| Moving Average | Smoothed price trend                | Identify direction          |
| Volatility     | Price fluctuation level             | Assess investment risk      |
